Início História

Nossa História

Como esse site foi criado do zero, num celular, em uma noite.

🚀 Passo a passo

1

Instalamos o Python

Usando o Termux no celular, instalamos o Python 3.13 e aprendemos os primeiros comandos.

2

Primeiros programas

Criamos uma calculadora completa e um quiz com alternativas usando if, elif e else.

3

Instalamos o Flask

Com pip install flask, criamos nosso primeiro servidor web rodando no próprio celular.

4

Construímos o site

Adicionamos HTML, CSS, JavaScript, contador de visitas e várias páginas.

5

Tudo no celular!

100% feito no Termux, sem computador, sem faculdade, só na raça! 🔥
