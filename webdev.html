🌐 Web Dev

Tudo que você precisa saber sobre desenvolvimento web moderno.

🤔 O que é Web Dev?

Web Development é a arte de criar sites e aplicações para a internet. É dividido em duas partes: o Frontend, que é tudo que o usuário vê e interage, e o Backend, que é a parte invisível que processa dados, faz login, salva informações e muito mais.

🎨 Frontend

O Frontend é a parte visual do site. É construído com HTML (estrutura), CSS (estilo e cores) e JavaScript (interação). Frameworks como React, Vue e Angular tornam o desenvolvimento mais rápido e organizado. É o que o usuário vê na tela.

HTMLCSSJavaScriptReactVue

⚙️ Backend

O Backend é o cérebro do site. Roda em servidores, processa dados, cuida do banco de dados e garante a segurança. É construído com linguagens como Python (Flask/Django), Node.js, Java ou PHP. Sem backend não há login, nem banco de dados.

PythonNode.jsFlaskDjangoPHP

🗄️ Banco de Dados

Todo site que salva informações precisa de um banco de dados. Os mais usados são MySQL, PostgreSQL e SQLite para dados relacionais, e MongoDB para dados não relacionais. Eles guardam usuários, posts, produtos e tudo mais.

MySQLPostgreSQLMongoDBSQLite
